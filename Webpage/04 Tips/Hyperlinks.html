超链接

this is a hyperlink to a picture

超链接中的索引对象和链接本身可以是字，文档，图片等；
使用href来确定跳转的对象 

        
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Hyperlink</title>
</head>
<body>
    <a href="敦煌奇妙夜.png">
        this is a hyperlink to a picture
    </a>
    <p>
        超链接中的索引对象和链接本身可以是字，文档，图片等；<br>
        使用href来确定跳转的对象
    </p>
</body>
</html>
