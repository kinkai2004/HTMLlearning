  • Apple

使用<ul>来实现无序列表；
type类型有disc,circle,square,none

disc 实心圆
circle 空心圆
square 方形
none 无
html源码： 
        
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=s, initial-scale=1.0">
    <title>Unordered listing</title>
</head>
<body>
    <ul type="circle">
        <li>
            Apple
        </li>
        <li>
            Banana
        </li>
        <li>
            Orange
        </li>
    </ul>
    <ul type="square">
        <li>
            Apple
        </li>
        <li>
            Banana
        </li>
        <li>
            Orange
        </li>
    </ul>
